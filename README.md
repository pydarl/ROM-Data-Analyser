# ROM Data Analyzer

A comprehensive data analysis and machine learning system for tracking rehabilitation outcomes and predicting Range of Motion (ROM) improvements in physical therapy patients.

## Overview

ROM Data Analyzer processes patient rehabilitation data to visualize progress, calculate statistics, and predict treatment outcomes using machine learning models. Built specifically for prosthetics and orthotics clinics, this tool helps therapists make data-driven decisions about patient care.

## Features

- **Data Processing**: Clean and analyze ROM measurements from CSV files
- **Interactive Visualizations**: Track patient progress over time with professional charts
- **Statistical Analysis**: Calculate improvement rates, treatment effectiveness, and outcome metrics
- **Machine Learning Predictions**: Predict ROM improvements based on patient demographics and treatment protocols
- **Multi-Model Comparison**: Compare Linear Regression, Decision Trees, and Random Forest models
- **Patient Dashboards**: Generate individualized progress reports for each patient

## Technologies Used

- **Python 3.9+**
- **Data Analysis**: Pandas, NumPy
- **Visualization**: Matplotlib, Seaborn
- **Machine Learning**: Scikit-learn
- **Development**: Jupyter Notebooks
